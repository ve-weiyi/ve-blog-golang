## 配置推荐

### 个人博客（日访问 < 1000）

```
CPU:    2核
内存:   2GB
硬盘:   40GB SSD
带宽:   1-3Mbps
价格:   约 60-100元/月
```

**适用场景**：个人博客、学习测试、小型项目

### 小型网站（日访问 1000-5000）

```
CPU:    2核
内存:   4GB
硬盘:   50GB SSD
带宽:   3-5Mbps
价格:   约 100-200元/月
```

**适用场景**：技术博客、小型社区、团队项目

```
价格:   约 300-500元/月
```

**适用场景**：商业网站、大型社区、企业应用

## 服务资源占用

### 基础服务内存需求

| 服务        | 最低内存  | 推荐内存  | 说明      |
|-----------|-------|-------|---------|
| MySQL 8.0 | 400MB | 512MB | 数据库服务   |
| Redis     | 50MB  | 128MB | 缓存服务    |
| RabbitMQ  | 128MB | 256MB | 消息队列    |
| Nginx     | 10MB  | 50MB  | Web 服务器 |
| Go 应用（单个） | 30MB  | 100MB | 后端服务    |

### 配置计算示例

**最小配置（2核2GB）**
```
MySQL (512MB) + Redis (128MB) + Go应用 (100MB) + 系统 (1GB)
= 约 1.74GB
剩余内存：260MB（较紧张）
```

**推荐配置（2核4GB）**

```
MySQL (512MB) + Redis (128MB) + RabbitMQ (256MB) + Go应用×3 (300MB) + 系统 (1GB)
= 约 2.2GB
剩余内存：1.8GB（充足）
```

## 购买流程

### 1. 注册账号

- 准备手机号和邮箱
- 完成实名认证（国内服务商必需）
- 绑定支付方式
- 领取新用户优惠券

### 2. 选择配置

**地域选择**

- 靠近目标用户所在地区
- 考虑网络延迟（可使用 ping 测试）
- 国内服务器需要备案

**操作系统推荐**

- CentOS 7/8（稳定，企业常用）
- Ubuntu 20.04/22.04（易用，社区活跃）
- Debian 11（轻量，稳定）

**镜像选择**

- 纯净系统镜像（推荐）
- 宝塔面板镜像（适合新手）
- Docker 镜像（容器化部署）

### 3. 安全组配置

**必须开放的端口**

| 端口  | 服务    | 说明        | 开放范围      |
|-----|-------|-----------|-----------|
| 22  | SSH   | 远程登录      | 限制 IP（推荐） |
| 80  | HTTP  | Web 服务    | 所有 IP     |
| 443 | HTTPS | 加密 Web 服务 | 所有 IP     |

**不建议开放的端口**

| 端口   | 服务       | 说明   | 建议          |
|------|----------|------|-------------|
| 3306 | MySQL    | 数据库  | 仅内网访问       |
| 6379 | Redis    | 缓存   | 仅内网访问       |
| 5672 | RabbitMQ | 消息队列 | 仅内网访问       |
| 9090 | API      | 应用接口 | 通过 Nginx 代理 |

### 4. 完成购买

- 确认配置和价格
- 选择购买时长（年付更优惠）
- 完成支付
- 记录服务器 IP 和初始密码

## 成本优化建议

- ✅ 从小配置开始，根据实际需求扩容
- ✅ 使用按量付费测试，确定需求后包年
- ✅ 使用 CDN 减少带宽消耗
- ✅ 使用对象存储存放静态资源
