# pkg 工具包使用说明

提供可复用的技术组件、工具函数和业务插件

## 📦 包结构

```
.
├── common/     # 常量和枚举
├── infra/      # 基础设施层
├── kit/        # 工具包和中间件
├── plugins/    # 功能插件
└── utils/      # 工具函数库
```

#### 📁 common/ - 通用业务组件

- **职责**: 存放跨服务的通用业务逻辑和数据结构
- **包含**: 通用模型、业务常量、共享业务规则
- **示例**: 用户模型、订单状态枚举、通用验证规则

#### 🏗️ infra/ - 基础设施层

- **职责**: 底层基础设施的抽象和实现
- **包含**: 数据库连接、配置管理、服务发现
- **示例**: MySQL连接池、配置加载器、注册中心客户端

#### 🔧 kit/ - 工具包和中间件

- **职责**: 提供可复用的技术组件和中间件
- **包含**: HTTP客户端、缓存组件、消息队列封装
- **示例**: Redis客户端、HTTP工具、日志中间件

#### 🛠️ utils/ - 工具函数库

- **职责**: 提供纯函数式的工具方法
- **包含**: 字符串处理、时间转换、加密解密
- **示例**: 字符串格式化、时间戳转换、MD5加密
